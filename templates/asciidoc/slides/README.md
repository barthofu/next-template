# Asciidoc - Slides

## Requirements

- [Node.js](https://nodejs.org/en/) >= 10.x.x
- [npm](https://www.npmjs.com/)

## Setup

```bash
npm install
```

## Usage

### Generate the HTML

```bash
npm run build
```

### Serve the slides

The slides are served on [http://localhost:8000/presentation.html](http://localhost:8000/presentation.html).

```bash
npm run serve
```
